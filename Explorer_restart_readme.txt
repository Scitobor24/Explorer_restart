Explorer Resurrection by Cooper
_______________________________
Dette er eit lite python-program som restartar Explorer.exe for å fikse diverse problemer, eg
laga dette fordi eg har ein eldre pc der eg ofte opplever at ikoner forsvinner, funksjonen til
programmet er der, men sjølve ikonet er vekke viss maskina har gått i lett dvale. 

Eg har prøvd mange metodar, alt frå å gå langt pokker i vold inn i kretser og chiper for
å løyse dette men eg kjem ingen veg. Så dette er ein slags last resource løysing.

Programmmet er laga i Python av "Cooper" aka Arnstein Austrheim og er delt som eit 
personlig prosjekt. Lisensen er fri og kan gjerast med kva ein vil.



Explorer Resurrection by Cooper

This is a small Python program that restarts Explorer.exe to fix various problems.
I created it because I have an older PC where I often experience that icons disappear.
The functionality of the program is still there, but the actual icon vanishes if the machine has been in light sleep.
I have tried many methods, even digging deep into circuits and chips to solve this,
but I got nowhere. So this is kind of a last‑resort solution.
The program is written in Python by "Cooper" aka Arnstein Austrheim and shared as a personal project.
The license is free, and you can do whatever you want with it.
